Please Note - This release of EXAFSPAK for Microsoft Windows requires an X Windows server package that must be installed before EXAFSPAK.

We have evaluated the following packages and verified that they work with EXAFSPAK but we do not endorse any particular package.

  • Cygwin provides a UNIX-like environment for Windows (available at no cost).
  • Hummingbird Exceed X Windows server - a full featured commercial package.
  • X-Win32 X Windows Server - a full featured alternative to Hummingbird.

Installation Procedure:

  1. Download the self-extracting .exe file by clicking here.
  2. Extract this file either to a location on your hard disk, an installation CD can now be created if desired. If a CD is used then this will auto-run upon insertion into your CD drive, and EXAFSPAK will be automatically installed. Otherwise double-click on setup.exe and the installation will begin.

If you are using Cygwin or X-Win32 (or any other package apart from Exceed) then you are now ready to run EXAFSPAK. Please refer to the manual for specific instructions on how to proceed.

If you are using the Hummingbird Exceed package then the following changes must be made to the system for EXAFSPAK to run optimally.

  1. Launch a command window and go to the EXAFSPAK installation directory by typing "cd %exafs%"
  2. Please make absolutely sure you are in the correct location before proceeding
  3. Delete all the .dll files by typing "del *.dll"
  4. Go to the system control panel, select the "advanced" tab, and click on "Environment Variables"
  5. Delete the system variable called "display"
  6. Edit the system variable called "path" and append the location of the Exceed installation at the end e.g. add
    ";C:\Program Files\Hummingbird\Connectivity\7.00\Exceed"
    • Note that this location depends upon the version of Exceed that is installed.

You have now completed your Windows EXAFSPAK installation.

Use the Command Prompt window to launch the programs in exactly the same way as for the UNIX and Open VMS versions.

Notes:

  1. If you have the Exceed services Telnetd and Xstartd enabled, then you will be able to use EXAFSPAK on your remotely using telnet to log in to your PC.
  2. EXAFSPAK default colors, fonts etc. can be controlled by changing the intitial settings given in the gplot.rc file in the home directory. An example gplot.rc file can be obtained by clicking here.
